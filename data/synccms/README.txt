backup
    시스템에서 자동으로 생성 된 파일 백업 삭제

dict
    사용자 정의 단어 분할 파일 디렉토리

history
   시스템에서 자동으로 생성되는 파일 수정 내역

indexes
    시스템에서 자동으로 생성되는 전체 텍스트 검색 색인 파일 디렉토리

task
    작업 예약 스크립트 디렉터리

template
    사이트 템플릿 디렉토리

web
    정적 파일 디렉토리

install.lock
    시스템에서 자동으로 생성되는 설치 잠금

database.properties
    데이터베이스 구성은 시스템에 의해 자동으로 생성됩니다. 여기서 jdbc.encryptPassword는 우선 순위가 jdbc.password보다 높은 암호화 된 데이터베이스 비밀번호입니다.
